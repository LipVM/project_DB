Данный проект представляет собой базу данных для мебельного магазина. База данных содержит информацию о товарах, клиентах, заказах и сотрудниках магазина.

Цели проекта:
- Автоматизация учёта товаров, клиентов, заказов и сотрудников магазина.
- Оптимизация процессов работы с данными.
- Повышение эффективности работы магазина.
- Структура базы данных

База данных состоит из следующих таблиц:

Товары(мебель): содержит информацию о товарах магазина (уникальный идентификатор товара, наименование, описание, цена, категория,количество на складе).
Клиенты: содержит информацию о клиентах магазина (уникальный идентификатор клиента, имя, адрес, телефон, email).
Заказы: содержит информацию о заказах клиентов (уникальный идентификатор заказа, ID клиента, сделавшего заказ, дата заказа, статус заказа).
Сотрудники: содержит информацию о сотрудниках магазина (уникальный идентификатор сотрудника, имя и фамилия, должность, дата найма).
Продажи: содержит информацию о прадажах магазина(уникальный идентификатор продажи, ID заказа к которому относится продажа, ID товара который был продан, количество проданного товара, сумма продажи).
Поставщики: содержит информацию о паставщиках (уникальный идентификатор поставщика, наименование поставщика, контактные данные поставщика, товары поставщика).
Отзывы клиентов: содержит информацию об отзывах (уникальный идентификатор отзыва, ID клиента оставившего отзыв, оценка данная клиентом например, от 1 до 5 звезд, содержание отзыва - текст отзыва).
Доставка: содеожит информацию о доставке заказов (уникальный идентификатор доставки, ID заказа к которому относится доставка, дата доставки когда товар был доставлен клиенту, статус доставки, ID сотрудника ответственного за доставку).
Возвраты товаров: содержит информацию о возвратах (уникальный идентификатор возврата, ID продажи к которой относится возврат, причина возврата товара, дата когда товар был возвращен).
